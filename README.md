# 幼儿园圣诞树生成系统

这是一个基于Python Flask的幼儿园圣诞树生成系统，具有管理员登录和圣诞树生成功能。

## 功能特点

1. **管理员登录**：使用指定账号(test)和密码(admin)登录系统
2. **圣诞树生成**：输入小朋友姓名生成专属圣诞树
3. **数据保存**：自动保存生成的圣诞树数据
4. **数据调取**：再次输入相同姓名时显示已保存的圣诞树
5. **童趣化设计**：色彩明快、符合幼儿审美的界面设计

## 技术栈

- **后端**：Python Flask
- **前端**：HTML、CSS、JavaScript
- **数据库**：SQLite

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问系统

打开浏览器访问：http://localhost:5000

## 使用说明

1. **登录系统**：使用账号test和密码admin登录
2. **生成圣诞树**：输入小朋友姓名，点击"生成圣诞树"按钮
3. **查看结果**：生成的圣诞树将显示在右侧区域
4. **退出登录**：点击页面顶部的"退出登录"按钮

## 数据库

系统使用SQLite数据库存储圣诞树数据，数据库文件`christmas_trees.db`会自动创建。

数据库表结构：
- `id`：主键，自增
- `child_name`：小朋友姓名（唯一）
- `tree_data`：圣诞树HTML数据
- `created_at`：创建时间

## 设计风格

- **色彩明快**：使用渐变色背景和明亮的颜色搭配
- **童趣化**：采用圆角设计、动画效果和可爱的装饰元素
- **布局清晰**：左右分栏布局，操作简单直观

## 开发说明

- Flask调试模式已开启，方便开发和调试
- 应用运行在0.0.0.0:5000，支持局域网访问
- 会话管理使用Flask内置的session功能
